Comment ouvrir un compte chez SafeBank ?

Pour ouvrir un compte, vous devez :

  • Une pièce d'identité valide
  • Un justificatif de domicile récent
  • Un dépôt initial minimum de 50.000 FC

Rendez-vous dans l'une de nos agences pour finaliser votre inscription.

Comment sécuriser mon compte en ligne ?

Voici les mesures recommandées :

  • Activez l'authentification à deux facteurs
  • Changez régulièrement votre mot de passe
  • Ne partagez jamais vos identifiants
  • Vérifiez régulièrement vos transactions

Comment utiliser l'application mobile ?

Pour utiliser l'application mobile :

  1. Téléchargez l'application sur votre store
  2. Connectez-vous avec vos identifiants
  3. Activez la reconnaissance biométrique
  4. Commencez à gérer vos comptes

Quels sont les frais de transfert ?

Nos frais de transfert :

  • Entre comptes SafeBank : Gratuit
  • Vers d'autres banques : 0.5%
  • Transferts internationaux : 1%
